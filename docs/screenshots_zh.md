# Brainhole 脑洞 - 界面截图说明

以下是 Brainhole 知识工作台的核心界面截图，展示了系统的主要功能与特性。

## 1. 画布编辑 (Canvas)

在画布中自由组合文本、表格、提示词、大模型等多种节点，构建可视化的 AI 工作流。

- **节点式工作流**：数据源 → 提示词 → AI 输出，通过可视化连线构建分析管道
- **拖拽操作**：从文件树或桌面直接拖入文件到画布，自动创建数据节点
- **灵活连线**：支持拖拽连线和点击连线两种交互方式，释放到节点卡片任意位置即可连接
- **智能创建**：从输出端口拖到空白处自动创建下游节点并连线
- **撤销/重做**：完整的操作历史记录
- **自动布局**：一键将节点重新排列为清晰的流程图
- **流式输出**：AI 结果实时渲染 Markdown，支持代码高亮、表格、Mermaid 图表
- **列表模式**：AI 输出可解析为列表，每项独立向下游节点传递

## 2. 文档编辑与阅读

支持 Markdown 富文本编辑，以及 PDF 原文预览与一键提取 Markdown。

- **所见即所得编辑**：基于 MDX Editor 的富文本 Markdown 编辑器，支持加粗、斜体、标题、列表等常用格式
- **代码与公式**：内嵌代码块语法高亮，支持 LaTeX 数学公式渲染
- **PDF 原文预览**：直接在应用内查看 PDF 原始排版，无需外部软件
- **一键提取 Markdown**：PDF 通过 MarkItDown / Docling / MinerU 三种引擎转换为结构化 Markdown
- **多格式支持**：PDF、Word、PPTX、XLSX、CSV、图片 (OCR)、Markdown、TXT 等

## 3. 知识图谱 (Knowledge Graph)

对文档进行实体抽取与关系推理，构建可交互的知识网络。

- **自动抽取**：AI 自动从文档中识别实体（人物、地点、概念等）并推理关系
- **力导向可视化**：基于 D3.js 力导向图交互式浏览，支持缩放、拖拽和节点聚焦
- **节点详情面板**：点击任意实体节点，查看详细属性、类型、关联度及描述
- **图谱问答 (GraphRAG)**：支持全局搜索和局部搜索两种 RAG 模式
- **自定义实体模板**：预置通用、保险、金融等领域模板，支持自定义实体类型

## 4. 多媒体资源处理

内置视频播放、音频提取与语音转录 (ASR) 功能。

- **视频浏览**：内置视频播放器，支持常见视频格式
- **音轨提取**：一键通过 ffmpeg 从视频中提取音频为独立文件
- **语音转文字 (ASR)**：基于 FunASR paraformer-zh 模型的高精度中文语音转录
- **AI 校对**：转录结果支持 AI 智能校对，可一键应用并替换原文
- **时间轴标注**：转录文稿自动标注时间段，便于定位和回溯

## 5. IMA 知识库

连接腾讯 IMA 知识库，接入外部知识源。

- **知识库浏览**：在侧边栏浏览个人知识库、共享知识库和订阅知识库的完整文件树
- **文件拖入画布**：笔记、PDF、Word、TXT 等文件可直接拖入画布作为数据源节点
- **内容自动获取**：笔记通过 API 获取原文，其他格式文件下载后自动本地解析

## 6. 全局菜单

左侧全局菜单提供搜索与最近访问等快捷入口。

- **全局搜索**：按文件名模糊检索知识库中的所有文档，快速定位目标文件
- **最近访问**：按时间倒序展示最近打开过的文档，一键跳转回溯
- **运行日志**：快速切换到日志面板，查看后台任务执行状态
- **系统设置**：一键打开设置面板，调整模型、引擎和偏好

## 7. 运行日志

实时查看后台任务的运行状态与输出日志。

- **虚拟滚动**：高性能渲染海量日志条目，数千条日志流畅滚动不卡顿
- **分类过滤**：可按「全部」「错误」「警告」分类筛选日志
- **多引擎日志**：同时展示 Docling、MinerU、FunASR 等多引擎的并行输出
- **一键清空**：快速清除历史日志，保持面板整洁
- **自动滚底**：新日志到达时自动滚动到底部，跟踪最新输出

## 8. 系统设置

可自由切换模型、解析引擎与图谱模板，满足深度定制需求。

- **AI 服务**：配置大语言模型 (LLM) 和向量模型 (Embedding) 的 Base URL、API Key 和模型名称，兼容所有 OpenAI 协议的 API
- **文档与图谱**：选择默认文档解析引擎（MarkItDown / Docling / MinerU），管理图谱实体类型模板（通用、保险、金融等领域）
- **通用**：界面语言切换（中文/English），后台任务并发数调节
- **自动保存**：所有配置修改立即生效并自动保存到本地，无需手动确认

## 9. 本地文件系统集成

与本地文件系统深度集成，提供完整的文件管理体验。

- **Vault 知识库**：选择本地任意文件夹作为知识库根目录，所有文件自动索引
- **文件夹分类**：支持自由创建文件夹层级，按工作、学习、资料库、创作等场景分类管理
- **多格式图标**：自动识别文件类型并显示对应图标（PDF、Markdown、音频、视频、表格、画布、图谱等）
- **文件导入**：通过原生文件选择器从系统任意位置导入文件到知识库
- **打开磁盘目录**：一键在系统文件管理器中定位并打开文件所在目录
